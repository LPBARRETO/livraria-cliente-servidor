# 📚 Livraria Digital - Arquitetura Cliente-Servidor

Este projeto separa as responsabilidades em duas camadas: um Backend (API FastAPI) e um Frontend independente.

## Como rodar o projeto:

1. **Ligar o Servidor:** 
   Abra o terminal na pasta raiz do projeto e rode o comando:
   `python run.py`

2. **Acessar o Cliente:**
   Navegue até a pasta `livraria_cliente` e dê dois cliques no arquivo `index.html` (ou rode o arquivo `cliente.py`, dependendo de qual você escolheu) para interagir com a API.
